01/05/2010 - Versão 2.0beta5:

	* A lista de siglas agora é geranda utilizando o programa makeindex fornecido
	juntamente com o MiKTeX. Veja o manual do pgeeltex para saber como utilizar
	o makeindex.
	* Adicionada opção de classe PGTEXdraft para gerar uma versão de impressão
	em página A4 (versão para a banca examinadora).
	* Adicionado comando \bancaEPGTEX para inserir mais um nome na folha de
	aprovação.



14/01/2010 - Versão 2.0beta4:

Mudanças na lista de siglas:
	* A lista é exibida em duas colunas, sendo que o tamanho da primeiracoluna é
	 relacionado com o tamanho da maior sigla.
	* O comando \sigla para inserir as siglas no texto automaticamente insere os
	 parenteses e não deixa mais espaços em branco.

Mudanças na lista de símbolos:
	* A lista é exibida em duas colunas, sendo que o tamanho da primeiracoluna é
	 relacionado com o tamanho do maior símbolo.
	* O comando \simb para inserir os símbolos no texto já insere automaticamente
	 em modo matemático, ou seja, no comando \simb não é necessária a utilização
	 de $$.

	Bug identificado:
	
	* Ao se utilizar o comando \simb dentro de um ambiente equation, a medida da
	 largura do símbolo não é corretamente efetuada, porém aparece normalmente 
	 na lista. Tome cuidado então, se o maior símbolo (mais largo) for definido
	 com o comando \simb dentro do equation, a lista pode ficar errada, com sobre-
	 posições de caracteres.


Referências em formato numérico:
	* Pode-se usar o formato de referências numéricas da abnt. Veja no código do
	 exemplo como proceder.
