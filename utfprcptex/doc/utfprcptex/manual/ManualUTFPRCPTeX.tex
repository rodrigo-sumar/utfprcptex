\chapter{Introdução}

O \utfprtex~ é uma classe de formatação para Dissertações, Trabalhos de Conclusão de Curso, Propostas de TCC e Trabalhos Acadêmicos da \ABNTinstituicaodata~ do \UTFPRunidadedata.

Por ser uma classe (modelo) o \utfprtex~ é invocado nas primeiras linhas do seu documento escrito no LaTeX, com o comando \verb|\documentclass|.

Este modelo é baseado no {\abnTeX} (http://abntex.codigolivre.org.br/)

Veja instruções abaixo de como instalar o \abnTeX~ bem como o \utfprtex.

\section{Pacotes Necessários a Execução do \utfprtex}

Além do pacote \abnTeX~, a instalação dos seguintes pacotes é necessária para a execução da classe \utfprtex:
\begin{itemize}
\item babel - Necessário para hifenização em portugues.
\item inputenc - Pacote para acentuação direta.
\item fontenc - Necessário para seleção da condificação das fontes.
\item array - Estende os ambientes tabular e array.
\item rotating - Rotacionar tabelas.
\item multirow - Permite a utilização de multiplas linhas em uma tabela.
\item colortbl - Permite colorir as linhas ou colunas das tabelas.
\item hhline - Permite melhor iteração das linhas duplas horizontais com as verticais.
\item ifthen - Efetuar teste do tipo se-senão
\item lastpage - Permitir a referencia a ultima pagina do documento.
\item perpage - Manter os contadores ordenados e permitir a numeração.
\item textcase - Conversão do texto para caixa alta.
\item fancyhdr - Pacote necessário para a geração da folha de aprovação.
\item xcolor - Utilizado para criar novas cores.
\item hyperref - Adicionar links ao documento.
\item nomencl - Pacote necessário para a lista de siglas.
\item caption - Pacote necessário para a formatação das legendas.
\item times - Selecionar fonte Times.
\item uarial - Selecionar fonte Arial.
\end{itemize}


\section{Instalação do \utfprtex }

Você pode instalar o \utfprtex de diversas maneiras, veja abaixo:

\subsection{A partir do download do arquivo compactado}

\begin{itemize}
	\item Baixe na página de download deste projeto o arquivo ''.zip´´ com a versão mais recente do \utfprtex. O endereço do site do \utfprtex~ é  \verb|http://code.google.com/p/pgeeltex/|
	\item Uma vez baixado o arquivo, descompacte-o em uma pasta exclusiva;
	\item Entre no menu de configurações (\textit{settings}) do MiKTeX (Menu iniciar/programas/MiKTeX 2.X);
	\item Na aba \textit{Roots} clique no botão \textit{Add} e adicione a pasta onde você descompactou o pgeeltex;
	\item Volte para a aba \textit{General} e clique nos botões \textit{Refresh FNDB} e \textit{Update Formats}.
\end{itemize}

Pronto, o MiKTeX já sabe onde está o \utfprtex.

\subsection{A partir do SVN}

Você pode também baixar o \utfprtex~ a partir do repositório SVN\footnote{Para mais informações sobre o SVN, veja em  (http://pt.wikipedia.org/wiki/Subversion)} deste projeto. A vantagem deste método é que você sempre baixará a versão mais recente do \utfprtex. Além disso, com um sistema de controle de versão\footnote{Veja a definição em  (http://pt.wikipedia.org/wiki/Sistema\_de\_controle\_de\_versão)} você pode facilmente atualizar o \utfprtex. O procedimento básico é o seguinte:

\begin{itemize}
	\item Baixe e instale um programa cliente SVN. Eu recomendo o Tortoise SVN que é integrado ao windows explorer:   \verb|http://tortoisesvn.tigris.org/|;
	\item Crie uma pasta para o \utfprtex~ no seu computador;
	\item Instado o Tortoise SVN clique com o botão direito do mouse dentro desta pasta e selecione \textit{SVN Checkout};
	\item Na janela que abriu, cole o seguinte endereço no campo \textit{URL of reposiroty}:

   \verb|https://utfprcptex.googlecode.com/svn/trunk/|

	\item Clique em \textit{OK} e espere o Tortoise conectar e baixar os diretórios e arquivos mais atuais do servidor;
	\item Pronto, agora você já tem a versão mais atual do \utfprtex;
	\item Entre no menu de configurações (\textit{settings}) do MiKTeX (Menu iniciar/programas/MiKTeX 2.X);
	\item Na aba \textit{Roots} clique no botão \textit{Add} e adicione a pasta onde você descompactou o \utfprtex;
	\item Volte para a aba \textit{General} e clique nos botões \textit{Refresh FNDB} e \textit{Update Formats}.

\end{itemize}

Pronto, o MiKTeX já sabe onde está o \utfprtex.

Se você quiser usar um cliente SVN na linha de comando do Windows (ou Linux), baixe os binários do SVN no site {http://subversion.tigris.org/} e após instalar, utilize a seguinte linha de comando:

\noindent
\verb|svn checkout http://utfprcptex.googlecode.com/svn/trunk/ utfprcptex-read-only|

\section{Instalação do AbnTeX}

Para que o \utfprtex~ funcione, é necessário que você tenha instalado o pacote \abnTeX. O procedimento é semelhante a instalação do \utfprtex~ via arquivo compactado:

\begin{itemize}
	\item Baixe o \abnTeX~ versão 0.8.2 em  \verb|http://abntex.codigolivre.org.br/|;
	\item Descompacte-o em uma pasta dedicada;
	\item Entre no menu de configurações (\textit{settings}) do MiKTeX (Menu iniciar/programas/MiKTeX 2.X);
	\item Na aba \textit{Roots} clique no botão \textit{Add} e adicione a pasta \textit{texmf} que está dentro da pasta onde você descompactou o \abnTeX;
	\item Volte para a aba \textit{General} e clique nos botões \textit{Refresh FNDB} e \textit{Update Formats}.
\end{itemize}

Pronto, agora o MiKTeX sabe onde está instalado o \abnTeX~ e você já pode começar a escrever seu trabalho!

As únicas diferenças estão na parte pré-textual do documento, na substituição das palavras referentes ao grau de formação do autor (mestre ou doutor) e na natureza do documento (dissertação ou tese).

\section{Comandos da classe}
\label{sec:comandos}


Alguns comandos adicionais foram definidos pela classe \utfprtex. De acordo com o tipo de documento (dissertação, tcc, trabalho,  etc.) a ser elaborado, o número de comandos necessários pode variar.
Todos os comandos definidos nesta classe tem em seu nome o sufixo UTFPR. Caso contrário, ou é um comando do \abnTeX~ ou um comando das classes padrões do \LaTeX.


Os comandos a seguir devem ser definidos no preâmbulo do documento. Ou seja, antes do comando \verb|\begin{document}|. Estes comandos servem para armazenar as informações relativas ao trabalho para que sejam utilizadas na criação e formatação dos elementos pré-textuais. A definição destes comandos é obrigatória de acordo com os elementos pré-textuais desta classe a serem utilizados (capa, folha de rosto, folha de aprovação, resumo e abstract).

\vspace{3ex}\noindent
\verb|\autor{nome}|

Insere \verb|nome| como sendo o nome do autor do texto. Utilizado na capa, folha de rosto, folha de aprovação, resumo e abstract.

\vspace{3ex}\noindent
\verb|\titulo{titulo do documento}|

Insere \verb|titulo do documento| como sendo o título da tese ou dissertação. Utilizado na capa, folha de rosto, folha de aprovação, resumo e abstract.

\vspace{3ex}\noindent
\verb|\title{document title}|

Insere \verb|document title| como sendo o título em inglês do trabalho, usando na criação do abstract. Utilizado somente no abstract.

\vspace{3ex}\noindent
\verb|\orientador[]{Grau}{Nome do orientador}|

Insere \verb|Nome do orientador| como sendo o nome do(a) orientador(a) do trabalho. O grau de formação deve ser indicado no respectivo campo. Possui um campo opcional (entre colchetes [ ]) em que pode ser modificada a palavra precedente ao nome do orientador. Se este campo não for preenchido, a palavra ``Orientador'' é utilizada. Se o orientador do trabalho for do sexo feminino, você deve preencher este campo com a palavra "Orientadora".

\vspace{3ex}\noindent
\verb|\coorientador[]{Grau}{Nome do co-orientador}|

Insere o nome do co-orientador do trabalho. Tem o mesmo funcionamento do comando \verb|\orientador|. Utilizado somente na folha de aprovação.

\vspace{3ex}\noindent
\verb|\coordenadorUTFPR[]{Grau}{Nome do coordenador}|

Insere o nome do coordenador do programa de pós-graduação. Tem o mesmo funcionamento do comando \verb|\orientador|. Utilizado somente na folha de aprovação.

\vspace{3ex}\noindent
\verb|\titulacao{titulacao}|

Insere a titulação a ser conferida, dada pelo texto \verb|titulacao| no argumento do comando. Utilizado nas folhas de rosto de de aprovação.

\vspace{3ex}\noindent
\verb|\area{area}|

Insere a área de concentração do trabalho, dada pelo texto \verb|area| no argumento do comando. Utilizado nas folhas de rosto de de aprovação.

\vspace{3ex}\noindent
\verb|\palavraschave{palavra 1, palavra 2, palavra 3, ... n}|

Insere a lista de palavras chave. Utilizado somente no resumo.

\vspace{3ex}\noindent
\verb|\keywords{keyword 1, keyword 2, keyword 3, ... n}|

Insere a lista de palavras chave em inglês. Utilizado somente no abstract.


\vspace{3ex}\noindent
\verb|\datadefesaUTFPR{dia/mes/ano}|

Insere data da defesa do trabalho dado pelo texto \verb|dia/mes/ano| no argumento do comando. É utilizado na folha de aprovação.

\vspace{3ex}\noindent
\verb|\data{ano}|

Insere o ano da defesa dado pelo texto \verb|ano| no argumento do comando. É utilizado na capa e folha de rosto.


\vspace{3ex}\noindent
\lstinline!\primeiroassina[Dr.]{Membro A da banca} ... \setimoassina[Dr.]{Membro E da banca}!

Este conjunto de comandos é utilizado para inserir o nome dos membros da banca de avaliação do trabalho. São utilizados somente pela folha de aprovação. Cada um dos comandos, \lstinline!\primeiroassina!, \lstinline!\segundoassina!, \lstinline!\terceiroassina!, \lstinline!\quartoassina!, \lstinline!\quintoassina!, \lstinline!\sextoassina! e \lstinline!\setimoassina! é utilizado para definir cada um dos membros da banca. Estes comandos possuem um argumento obrigatório, dado pelo texto \verb|Membro X da banca| que corresponde ao nome da pessoa. O grau de titulação de cada membro pode ser informado pelo campo opcional entre colchetes. Se este campo não for especificado, a palavra ``Doutor'' é utilizada como padrão.


\vspace{3ex}\noindent
\lstinline!\numerodemembrosnabanca{numero de membros}!

Este comando é utilizado para definir se os campos de assinaturas dos membros da banca na folha de aprovação serão colocados em uma ou duas colunas. Se o orientador e/ou o coorientador não fizerem parte da banca eles não deverão ser considerados no numero total de membros.

\vspace{3ex}\noindent
\lstinline!\orientadornabanca{sim ou nao}!

Este comando é utilizado para definir se o campo de assinatura do orientador será inserido entre os membros da banca ou não. No caso de o orientador não participar da banca sua assinatura constará junto a do coordenador do curso.

\vspace{3ex}\noindent
\lstinline!\coorientadornabanca{sim ou nao}!

Este comando é utilizado para definir se o campo de assinatura do coorientador será inserido entre os membros da banca ou não. No caso de o coorientador não participar da banca sua assinatura constará junto a do coordenador do curso.

\vspace{3ex}\noindent
\verb|\textoaprovacao{texto}|

Insere texto a ser utilizado na folha de aprovação do trabalho.

\vspace{3ex}\noindent
\verb|\local{local}|

Insere o local de apresentação do  trabalho. Utilizado na capa, folha de rosto e folha de aprovação.


\vspace{3ex}\noindent
\verb|\comentario{comentario}|

Insere o comentário a ser apresentado na folha de rosto.


\vspace{3ex}\noindent
\verb|\cita{SOBRENOME, Nome}|

Insere a citação do autor do trabalho para ser utilizado no resumo e abstract.

\verb|\documento{documento}|


Insere o tipo do documento (Trabalho de Conclusão, Dissertação...). Utilizado na capa, folha de rosto, folha de aprovação, resumo e abstract.


\vspace{3ex}\noindent
\verb|\unidade{unidade}|

Insere a unidade da instituição a qual pertence o curso. Utilizado na capa. É um comando opcional.


\vspace{3ex}\noindent

\vspace{3ex}\noindent
\verb|\autordois{nome} ... \autorquatro{nome}|

Este conjunto de comandos é utilizado para inserir o nome dos autores no caso de elaboração de trabalhos acadêmicos ou trabalhos de conclusão de curso. Para a capa, os comandos utilizados são:\lstinline!\instituicao{}!, \lstinline!\unidade{}!, \lstinline!\departamento{}!, \lstinline!\programa{}!, \lstinline!\documento{}!, \lstinline!\autor{}!, \lstinline!\titulo{}!, \lstinline!\data! e \lstinline!\local{}!.

\subsubsection{Capa Pré-Proposta}
O comando \verb|\capapre| é utilizado para mostrar a capa do documento, de acordo com o padrão da UTFPR. Para que a capa seja criada corretamente, algumas informações devem ser fornecidas pelo usuário, como título, nome do autor, etc. Estas informações são fornecidas através dos comandos, apresentados na Seção \ref{sec:comandos}, os quais devem ser definidos no preâmbulo do documento.

Para a capa, os comandos utilizados são:\lstinline!\instituicao{}!, \lstinline!\unidade{}!, \lstinline!\departamento{}!, \lstinline!\programa{}!, \lstinline!\documento{}!, \lstinline!\autor{}!, \lstinline!\titulo{}!, \lstinline!\data! e \lstinline!\local{}!.

\subsection{Folha de rosto}

O comando \verb|\folhaderosto[opção]| é utilizado para mostrar a folha de rosto do documento. Para isso, os seguintes comandos devem ser definidos no preâmbulo do documento: \verb|\autor{}|, \verb|\comentario{}|, \verb|\titulo{}|, \verb|\orientador[]{}{}|, \verb|\coorientador[]{}{}|, \verb|\data{}| e \verb|\local{}|. O parâmetro opcional serve para indicar se a folha de rosto será gerada com ou sem ficha catalográfica.

\subsubsection{Ficha Catalográfica}
No caso de se utilizar o comando \verb|\folhaderosto[opção]| com a opção de gerar a ficha catalográfica, ao se compilar o documento a classe irá buscar um arquivo nomeado {\it fichacatalografica.tex} o mesmos diretório em que se encontra o arquivo \TeX~ principal do trabalho. Portanto, ao se optar pela ficha catalográfica, tal arquivo deve existir.

O arquivo disponibiliado no modelo faz uso dos seguintes comandos, que dever ser definidos: \verb|\abreviatura{sigla}|,  \lstinline!\assuntos{Assunto 1}{Assunto 2}{Assunto 3}{Assunto 4}!, \verb|\cutter{XXXXx}|, \verb|\cdd{CDD-XXX.X}|.


\subsection{Folha de aprovação}

A folha de aprovação é mostrada com o comando \verb|\termodeaprovacao|. Este por sua vez, depende dos comandos \verb|\autor{}|, \verb|\titulo{}|, \verb|\textoaprovacao{}|, \verb|\datadefesaUTFRP{}|, \verb|\coordenadorUTFPR[]{}{}|, \verb|\orientador[]{}{}|, \lstinline!\coorientador[]{}{}! (se houver) e dos comandos dos membros da banca dados por \lstinline!\primeiroassina!, \lstinline!\segundoassina!, \lstinline!\terceiroassina!, \lstinline!\quartoassina!, \lstinline!\quintoassina!, \lstinline!\sextoassina! e \lstinline!\setimoassina! de acordo com seu número. Adicionalmente tem-se os comandos \lstinline!\numerodemembrosnabanca{numero de membros}!, \lstinline!\orientadornabanca{sim ou nao}! e \lstinline!\coorientadornabanca{sim ou nao}!.

As instruções para a organização da banca examinadora são as seguintes:
\begin{itemize}
	\item O primeiro membro da banca, definido pelo comando \verb|\orientador| é o presidente da banca;
	\item O segundo membro será automaticamente o Co-orientador se ele existir e for definido pelo comando \verb|\coorientador|;
	\item Os demais membros da banca são definidos pelos comandos \lstinline!\primeiroassina!, \lstinline!\segundoassina!, \lstinline!\terceiroassina!, \lstinline!\quartoassina!, \lstinline!\quintoassina!, \lstinline!\sextoassina! e \lstinline!\setimoassina!;
	\item Deixando os parâmetros dos comandos dos membros da banca vazios (como por exemplo \verb|\primeiroassina{}|), as respectivas linhas de assinatura não aparecerão;
	\item Os comandos suportam um parâmetro adicional, entre colchetes, onde é colocado o grau de titulação do respectivo membro da banca. Se este parâmetro adicional não for utilizado, então o valor padrão \textit{Doutor} é atribuído.
\end{itemize}

\subsection{Lista de Siglas e Abreviações}

Você pode utilizar o \utfprtex\ para gerar uma lista de siglas e abreviações de forma automática. O método descrito aqui é baseado no pacote \verb|nomencl|. Se você tiver mais dúvidas, pode consultar o manual deste pacote \cite{nomencl}.

Para gerar a lista de siglas no \utfprtex você deve fazer o seguinte:

\begin{enumerate}
	\item Inserir os seguintes comandos no preâmbulo do seu arquivo .tex: \verb|\usepackage{nomencl}| e logo em seguida \verb|\fazlistasiglas|. Este último aceita uma parâmetro opcional utilizado para redefinir o título da lista de siglas. O texto padrão para o título é ``Lista de Siglas'';
	\item Inserir o comando \verb|\listadesiglas| no ponto do texto onde você quer que a lista de siglas apareça. Por exemplo, após a lista de tabelas;
	\item Na primeira vez que você for definir uma sigla no texto, utilize o comando \verb|\sigla|. Este comando gera uma nova entrada na lista de siglas. Dois parâmetros são necessários para ele: o significado da sigla e a sigla em si. Por exemplo:
	
	\verb|\sigla{ANEEL}{Agência Nacional de Energia Elétrica}|
	
	Este comando irá inserir automaticamente o significado no texto e a sigla entre parênteses. Se você não quiser que a sigla apareça entre parênteses no texto, utilize a versão \textit{estrelada} do comando: \verb|\sigla*{sigla}{descrição}|.

\item Para que a lista de siglas apareça, o programa \textit{makeindex} deve ser rodado após rodar o Latex e em seguida rodar o Latex novamente. A linha de comando do \textit{makeindex} é a seguinte:
	
	\verb|makeindex.exe nome_arquivo.nlo -s nomencl.ist -o nome_arquivo.nls|
	
	Onde \verb|nome_arquivo| é o nome do seu arquivo principal do documento, sem a extensão \verb|.tex|.
    A classe \utfprtex\ possui um comando interno que executa esse processo automaticamente.
	Se você utiliza o Texniccenter como editor, configure-o para rodar o \textit{makeindex} automaticamente a cada vez que você compila o seu projeto apenas apertando a tecla F7. Veja a seguir como fazer isso.	
\end{enumerate}

Configurando o texniccenter para rodar o \textit{makeindex}:

\begin{enumerate}
	\item Vá no menu Build/Define output profile;
	\item Clique no profile LaTeX $\Rightarrow$ DVI;
	\item Desmarque a caixa de seleção \textit{Do not use makeindex in this profile};
	\item Nos campo abaixo, se já não estiver configurado, indique o caminho onde está o textit{makeindex.exe};
	\item No campo dos argumentos a serem passados para o \textit{makeindex} coloque o seguinte:
	
	\verb|"%bm".nlo -s nomencl.ist -o "%bm".nls|
	
	\item No menu Project/Properties do texniccenter, marque a caixa de seleção \textit{Uses Makeindex}.
\end{enumerate}

Pronto, assim que você compilar o seu projeto no Texniccenter, o mesmo se encarregará de rodar makeindex. Você pode fazer a mesma coisa em outros \textit{profiles} também.

A largura das colunas da lista de siglas é calculada de acordo com a largura da maior sigla presente no texto. Este valor fica armazenado em um arquivo com extensão \verb|.tmp| na pasta do seu arquivo tex principal. Se você quiser \textit{resetar} o tamanho máximo, apague este arquivo e rode novamente o latex.

Veja mais exemplos da lista de siglas no documento exemplo que acompanha o \utfprtex.



\subsection{Lista de Símbolos}

A lista de símbolos funciona de forma semelhante a um sumário (\textit{table of contents}). Os símbolos aparecem na ordem em que foram definidos no texto.

Para gerar a lista de símbolos no \utfprtex\ você deve fazer o seguinte:


\begin{enumerate}
	\item Inserir o comando \verb|\listadesimbolos| no ponto do texto onde você quer que a lista de símbolos apareça. Por exemplo, após a lista de siglas;
	\item Na primeira vez que você for definir um símbolo no texto, utilize o comando \verb|\simbolo|. Este comando gera uma nova entrada na lista de símbolos. Dois parâmetros são necessários para ele: o significado da símbolo e o símbolo em si, utilizando a seguinte sintaxe: \verb|\simbolo{símbolo}{descrição}|.	
\end{enumerate}

Não use o comando \verb|\simbolo| dentro de um ambiente de equações como o \verb|equation|, pois neste caso a largura do símbolo para formatação da lista de símbolos não será calculada corretamente e poderão haver sobreposições de caracteres.

Veja mais exemplos da lista de símbolos no documento exemplo que acompanha o \utfprtex.



\section{Elementos pós-textuais}

Elementos pós textuais, como anexos e apêndices podem ser inseridos utilizando as definições da classe \abnTeX~ \cite{abntex-doc}.

Utilize os comandos \verb|\anexo| e \verb|\apendice| para dar início aos anexos e apêndices, respectivamente. Após estes comandos, você pode utilizar normalmente o \verb|\chapter{}|, \verb|\section{}|, etc. para definir capítulos e seções nos anexos e/ou apêndices.

Para maiores detalhes, leia a documentação do \abnTeX.
